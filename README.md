## Project

This is the final fullstack project of Skylab Coders Academy course.

## Technologies

In front:

-   React
-   Flux
-   Auth0
-   Material UI
-   Google Map React
-   Jest
-   Axios

In back:

-   Express
-   Mocha
-   Chai
-   Sinon

## Setup

To run this project install it locally first in backend and then in frontend using npm:

```
$ npm install
$ npm start
```
